Overview

This project focuses on analyzing customer demographic data to understand patterns in age, gender, region, and behavior. The goal of this dashboard is to help businesses make better, data-driven decisions by presenting insights in a clear and interactive way.

The dashboard was built to simplify complex data and highlight trends that can support marketing, customer segmentation, and strategic planning.

Tools & Technologies Used:-

-Power BI

-Power Query

-DAX

-SQL

-Excel

Dataset Description:-

The dataset includes customer-level information such as demographic details, behavioral attributes, and transactional data.
Data cleaning and transformation were performed to ensure accuracy and consistency before building the dashboard.

Key Insights:-

Identified major customer segments based on demographic attributes

Highlighted patterns in customer distribution across different regions

Improved understanding of customer behavior to support targeted marketing

Helped visualize trends that can improve decision-making efficiency

Dashboard Features:-

Interactive filters for easy exploration

Clear visualizations for demographics and trends

How to Use:-

Download the .pbix file

Open it using Power BI Desktop

Explore the dashboard using slicers and filters
